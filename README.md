

## project - Simple Banking System By Avni jain..

## Overview
Console-based Simple Banking System developed using Java.
Features:
- Deposit Money
- Withdraw Money
- Check Balance

## How to run
1. Open terminal in src folder
2. javac BankingApp.java
3. java BankingApp
